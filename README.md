# Library Project

A personal library web app, with a simple design.

## Overview

This project was completed as part of The Odin Project full stack development course. It is built with HTML, CSS, and JS. 

It lets the user add and remove books from their personal library, and using the unique ISBN reference for the book, automatically fetches the correct book cover image. 

Live demo: [link] (https://blakkmango.github.io/library-top/)

It does not have any type of storage at present as this was outside the scope of this project.

### Skills practiced
- Creating book objects with a constructor
- Storing books inside an array
- Rendering dynamic content on the page
- Using event delegation for remove and mark as read features
- Handling form submission with `event.preventDefault()`
- Creating, opening, and closing modals
- Form creation
